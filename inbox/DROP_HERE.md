# INBOX - שים כאן חומר חדש

## איך זה עובד:
1. תזרוק לכאן כל קובץ חדש (וידאו, תמונה, אודיו, מסמך)
2. תגיד לי "סווג את ה-inbox" או "תעבד את התיקייה"
3. אני אנתח כל קובץ, אסווג אותו, ואעביר למקום הנכון

## מה אני עושה עם כל קובץ:
- **וידאו B-Roll** → `content/raw/broll/[קטגוריה]/`
- **Talking Head** → `content/raw/talking-heads/`
- **Screen Recording** → `content/raw/screen-recordings/`
- **תמונות** → `content/raw/photos/`
- **הקלטות קול** → `content/voice_samples/`
- **סרטון לאווטר** → `content/avatar/`
- **מסמכים** → `docs/`
