# תסריט הקלטה - "העוזר הראשון שלך" 🎤

> **אורך כולל:** ~30 דקות
> **פורמט:** קריאה להקלטה (voiceover + screen recording)
> **סגנון:** קז'ואל, ישיר, כמו חבר שמסביר
> **הערה:** [מסך] = חלקים שילוו בscreen recording. [מצלמה] = פנים למצלמה.

## שיעור 1: "למה אתה צריך עוזר AI" (4 דקות)

### [מצלמה - פנים למצלמה]

אוקיי, אז לפני שנתחיל - אני רוצה להראות לך משהו.

היום בבוקר, קמתי, הכנתי קפה, ותוך כדי שהילדים שלי אכלו ארוחת בוקר - קיבלתי פודקאסט חדשות מותאם אישית. לא הלכתי לשום אתר. לא פתחתי שום אפליקציה. עוזר AI שבניתי הכין לי את זה לבד. כל בוקר. אוטומטי.

אותו עוזר? גם אמר לי מה האימון שלי היום, מה לקנות בסופר לארוחת הערב, ומה המשימות הכי דחופות שמחכות לי בעסק.

כל זה? בלי שנגעתי בשום דבר.

ואני לא מתכנת. אני בן אדם שבנה עסק סולארי, שירת ביחידה מיוחדת, ועכשיו חי עם המשפחה שלי בקו פנגאן, תאילנד.

אם אני יכול - גם אתה יכול. ובחצי שעה הקרובה, אתה תבנה את העוזר הראשון שלך.

### [מצלמה]

אז מה זה בכלל "עוזר AI"?

תחשוב על זה ככה: יש לך עובד. הוא זמין 24 שעות. הוא לא מתלונן. הוא לא שוכח. הוא לא צריך חופשה. והוא טוב ב... בעצם כמעט בכל דבר שקשור למחשב.

כתיבה? הוא מעולה. מחקר? מעולה. ניתוח נתונים? מעולה. ארגון? מעולה. תכנון ארוחות, אימונים, לוחות זמנים, סיכומי פגישות? מ-עו-לה.

והדבר הכי חשוב - הוא לא צריך ממך שום ידע טכני. אתה מדבר איתו. בעברית. במילים רגילות. והוא עושה.

### [מצלמה]

עכשיו, יש הרבה כלי AI בחוץ. ChatGPT, Gemini, Copilot... אני בחרתי לעבוד עם Claude, של חברת Anthropic. ויש לזה סיבה.

Claude מבין הקשר טוב יותר. כשאתה נותן לו מידע על מי אתה, מה העסק שלך, מה המטרות שלך - הוא זוכר את זה ועובד אתך לאורך זמן. הוא לא סתם עונה על שאלות. הוא באמת עוזר.

וב-30 הדקות הקרובות, אני הולך ללמד אותך:
1. איך להתקין את Claude על המחשב שלך
2. איך לדבר איתו נכון ככה שיתן לך תשובות מדויקות
3. ואיך לבנות את העוזר הראשון שלך - אחד שיחסוך לך לפחות שעה ביום

בואו נתחיל.

## שיעור 2: "התקנת Claude על המחשב" (7 דקות)

### [מצלמה]

אוקיי, שיעור 2 - התקנה. זה לוקח 5 דקות. אין מה לפחד.

יש שתי דרכים להשתמש ב-Claude:
1. דרך האתר - claude.ai - בדפדפן, בלי להתקין כלום
2. דרך Claude Code - ישירות על המחשב שלך, דרך הטרמינל

אנחנו הולכים להתקין את שתיהן. כי האתר טוב לשיחות מהירות, ו-Claude Code? זה שם שהקסם האמיתי קורה.

### חלק A: האתר (2 דקות)

### [מסך - מראה דפדפן]

ראשית, האתר. נכנסים ל-claude.ai.

לוחצים "Sign Up". אפשר להירשם עם גוגל, עם אימייל - מה שנוח. עושים את זה עכשיו, אני מחכה.

מעולה. ברגע שנכנסת - אתה רואה את הצ'אט. זה Claude. אפשר לדבר איתו. תכתוב "היי, מה שלומך?" ותראה שהוא עונה. בעברית. מצוין.

הגרסה החינמית עובדת מעולה להתחלה. אם תרצה יותר הודעות ביום ומודל חזק יותר, יש Pro ב-20 דולר לחודש. אבל בינתיים? חינם מספיק.

### חלק B: Claude Code (5 דקות)

### [מצלמה]

עכשיו - Claude Code. וזה החלק שאולי נשמע מפחיד. "טרמינל"? "שורת פקודה"?

תירגע. אני הולך להראות לך בדיוק מה ללחוץ. צעד אחרי צעד.

### [מסך - Mac]

**למשתמשי Mac:**

שלב 1: פותחים את Terminal. איפה זה? לוחצים Command + Space, כותבים "Terminal", לוחצים Enter. זהו, נפתח חלון שחור. אל תיבהל, זה בסדר.

שלב 2: מתקינים Node.js. זו תוכנה שClaude Code צריך כדי לרוץ. מעתיקים את הפקודה הזו ומדביקים בטרמינל:

```
curl -fsSL https://fnm.vercel.app/install | bash && fnm install --lts
```

לוחצים Enter. מחכים שיסיים. לוקח חצי דקה.

שלב 3: מתקינים את Claude Code עצמו:

```
npm install -g @anthropic-ai/claude-code
```

Enter. מחכים. עוד חצי דקה.

שלב 4: מפעילים:

```
claude
```

בום. Claude Code רץ על המחשב שלך. הוא יבקש ממך להתחבר - לוחצים על הלינק, מאשרים, וזהו.

### [מסך - Windows]

**למשתמשי Windows:**

שלב 1: פותחים PowerShell. לוחצים על כפתור Start, כותבים "PowerShell", לוחצים Enter.

שלב 2: מתקינים Node.js. נכנסים לאתר nodejs.org, מורידים את הגרסה של Windows, מתקינים כמו כל תוכנה - Next, Next, Finish.

שלב 3: סוגרים את PowerShell ופותחים מחדש. ואז:

```
npm install -g @anthropic-ai/claude-code
```

Enter. מחכים.

שלב 4:

```
claude
```

ומתחברים.

### [מצלמה]

מעולה. אם הגעת לפה - יש לך את Claude על המחשב. ועכשיו מגיע החלק הכיפי.

## שיעור 3: "איך לדבר עם Claude (ולקבל תשובות מדויקות)" (6 דקות)

### [מצלמה]

רוב האנשים משתמשים ב-AI כמו גוגל. שואלים שאלה, מקבלים תשובה, הולכים. וזה בזבוז של 90% מהפוטנציאל.

Claude הוא לא מנוע חיפוש. הוא עוזר. וכמו כל עוזר טוב - ככל שהוא מכיר אותך יותר, ככה הוא עוזר יותר טוב.

אז בואו נלמד שני דברים:
1. איך לשאול נכון (prompting)
2. איך לתת לClaude להכיר אותך (CLAUDE.md)

### חלק A: לשאול נכון (3 דקות)

### [מסך - claude.ai פתוח]

הנה הכלל הכי חשוב: **תן הקשר.**

דוגמה גרועה:
> "תכתוב לי מייל"

דוגמה טובה:
> "אני בעל עסק קטן בתחום הייבוא. אני צריך לכתוב מייל לספק בסין שמאחר במשלוח. הטון צריך להיות מקצועי אבל נחרץ. המשלוח מאחר ב-3 שבועות."

תראה את ההבדל? בדוגמה הראשונה Claude מנחש. בשנייה - הוא יודע בדיוק מה אתה צריך.

### [מסך]

הנה הנוסחה. כל פעם שאתה מבקש משהו מClaude, תן לו:

1. **מי אתה** - "אני בעל עסק..." / "אני מנהל צוות של..." / "אני עצמאי ב..."
2. **מה אתה צריך** - "אני צריך..." / "תעזור לי עם..." / "תכין לי..."
3. **איך אתה רוצה את זה** - "בטון מקצועי" / "בעברית פשוטה" / "בפורמט רשימה"
4. **הקשר נוסף** - "הלקוח שלי הוא..." / "התקציב הוא..." / "הדדליין הוא..."

ככל שתיתן יותר הקשר - התשובה תהיה יותר מדויקת. תמיד.

### [מסך - מראה דוגמה חיה]

בואו ננסה. אני כותב לClaude:

> "אני בעל עסק קטן עם 3 עובדים. אני מבזבז שעה ביום על לענות על אימיילים של לקוחות. רובם שואלים את אותן 10 שאלות. תעזור לי לבנות מסמך FAQ שאני יכול לשלוח ללקוחות במקום לענות כל פעם מחדש. העסק שלי עוסק ב[תחום שלך]."

תראה מה הוא מחזיר... מדהים, נכון? FAQ מסודר, מקצועי, מותאם לעסק שלך. עכשיו תעשה copy-paste, תתאים קצת, ויש לך מסמך שהיה לוקח לך שעתיים לכתוב. נעשה ב-30 שניות.

### חלק B: CLAUDE.md - הזיכרון של Claude (3 דקות)

### [מצלמה]

עכשיו - הטריק שאף אחד לא מלמד.

CLAUDE.md. זה קובץ טקסט פשוט שאתה שם על המחשב שלך. וכל פעם שClaude Code עולה - הוא קורא אותו. זה כמו ה-CV שלך בשבילו. הוא יודע מי אתה, מה העסק שלך, מה המטרות שלך, מה אתה אוהב ומה לא.

### [מסך - פותח קובץ טקסט]

בואו ניצור אחד עכשיו.

פותחים כל תוכנה לעריכת טקסט. Notes, Notepad, מה שיש. ושומרים קובץ בשם CLAUDE.md בתיקייה הראשית שלכם.

בפנים כותבים:

```markdown
# מי אני
- שם: [השם שלך]
- עיסוק: [מה אתה עושה]
- עסק: [תיאור קצר של העסק]

# מה חשוב לי
- [דבר 1]
- [דבר 2]
- [דבר 3]

# איך אני אוהב לקבל תשובות
- בעברית פשוטה
- עם דוגמאות מעשיות
- בפורמט רשימה כשאפשר

# המטרות שלי
- [מטרה 1]
- [מטרה 2]
```

### [מצלמה]

זהו. פשוט ככה. מלאת? מעולה.

מעכשיו, כל פעם שתפתח Claude Code בתיקייה הזו - הוא כבר יודע מי אתה. לא צריך להסביר שוב ושוב. הוא כבר "מכיר" אותך.

וזה ההבדל בין להשתמש ב-AI לבין שיש לך **עוזר אישי** שמכיר אותך.

## שיעור 4: "בנה את העוזר הראשון שלך" (10 דקות)

### [מצלמה]

אוקיי, הגענו לחלק הכי מגניב. עכשיו אנחנו בונים.

אני הולך להראות לך 3 סוגי עוזרים. תבחר אחד - זה שהכי רלוונטי לך עכשיו - ונבנה אותו ביחד. בסוף השיעור הזה, יהיה לך עוזר AI שעובד.

שלוש האפשרויות:
- **עוזר בוקר** - מכין לך סיכום חדשות + סדר יום + משימות
- **עוזר כושר** - תכנון אימונים + תזונה + מעקב
- **עוזר עסקי** - ניהול משימות + כתיבת מיילים + ארגון

תבחר. בחרת? יופי. אני אראה את כולם, ואתה תעקוב אחרי זה שבחרת.

### אפשרות A: עוזר בוקר (3 דקות)

### [מסך - Claude Code פתוח]

פותחים את Claude Code. כותבים:

```
claude
```

ועכשיו, מדברים איתו. ככה:

> "אני רוצה שתהיה העוזר הבוקרי שלי. כל בוקר אני רוצה שתכין לי:
>
> 1. סיכום של 5 הכותרות הכי חשובות מישראל והעולם
> 2. מזג האוויר ב[עיר שלך]
> 3. רשימת 3 המשימות הכי חשובות שלי להיום
> 4. תזכורת לדבר אחד שאמרתי שאני רוצה לעשות השבוע
>
> תציג את הכל בפורמט נקי וקצר. בעברית."

### [מסך - מראה את התשובה]

תראה מה הוא עשה. בום. סיכום חדשות, מזג אוויר, משימות - הכל מסודר, הכל בעברית.

עכשיו, הנה הטריק: אתה יכול לשמור את ההוראה הזו. כל בוקר פשוט פותח Claude Code ואומר: "בוקר טוב, תכין לי את הסיכום היומי" - והוא כבר יודע מה לעשות, כי ה-CLAUDE.md שלך מגדיר את זה.

בואו נוסיף את זה ל-CLAUDE.md:

```markdown
# שגרת בוקר
כל בוקר כשאני אומר "בוקר טוב" או "סיכום בוקר":
- 5 כותרות חדשות מישראל ומהעולם
- מזג אוויר ב[עיר]
- 3 משימות עיקריות להיום
- תזכורת שבועית
```

שמרתם? מעכשיו יש לכם עוזר בוקר.

### אפשרות B: עוזר כושר (3 דקות)

### [מסך - Claude Code פתוח]

לעוזר הכושר, כותבים:

> "אני רוצה שתהיה המאמן האישי שלי. הנה המידע עליי:
> - גובה: [X] ס"מ
> - משקל: [X] קילו
> - מטרה: [לרזות / לעלות מסה / כושר כללי / להתחזק]
> - ניסיון: [מתחיל / בינוני / מתקדם]
> - ציוד זמין: [חדר כושר / בית / משקולות / כלום]
> - זמן לאימון: [X] דקות, [X] פעמים בשבוע
>
> תבנה לי תוכנית אימונים לשבוע, ותגיד לי מה לאכול היום."

### [מסך - מראה תשובה]

תראה - תוכנית אימונים מלאה. מותאמת אישית. עם סטים, חזרות, זמני מנוחה. ותפריט תזונה ליום.

נוסיף ל-CLAUDE.md:

```markdown
# כושר ותזונה
- גובה: [X], משקל: [X], מטרה: [X]
- מתאמן [X] פעמים בשבוע, [X] דקות
- ציוד: [מה שיש]
- אלרגיות/העדפות אוכל: [X]
כשאני אומר "אימון" - תן לי את האימון של היום
כשאני אומר "מה לאכול" - תן לי תפריט מותאם
```

### אפשרות C: עוזר עסקי (4 דקות)

### [מסך - Claude Code פתוח]

העוזר העסקי - זה שחוסך הכי הרבה זמן. כותבים:

> "אני רוצה שתהיה העוזר העסקי שלי. הנה המידע על העסק:
> - תחום: [מה העסק]
> - גודל: [כמה עובדים / עצמאי]
> - לקוחות: [מי הלקוחות]
> - אתגר עיקרי: [מה הכי מציק לך בעסק]
>
> עכשיו, תעזור לי עם הדבר הבא:
> [בחר אחד]
> - תכתוב לי מייל מקצועי ל[מי]
> - תסכם לי את הפגישה מהתקליט הזה: [הדבק תקציר]
> - תארגן לי רשימת משימות לשבוע לפי עדיפות
> - תנתח לי את המכירות מהחודש האחרון
> - תכין לי הצעת מחיר ל[פרויקט]"

### [מסך - מראה תשובה]

תראה? מייל מקצועי. מותאם. בטון שלך. ב-10 שניות.

עכשיו תוסיף ל-CLAUDE.md:

```markdown
# העסק שלי
- תחום: [X]
- לקוחות: [X]
- שירותים: [X]
- טון תקשורת: [מקצועי / ידידותי / רשמי]

כשאני אומר "מייל" - תעזור לי לכתוב מייל עסקי
כשאני אומר "משימות" - תסדר לי עדיפויות
כשאני אומר "הצעת מחיר" - תעזור לי לבנות אחת
```

### [מצלמה]

שמעת? שלושה עוזרים. בחרת אחד. בנית אותו. זה עובד.

ועכשיו הנקודה הכי חשובה: **תשתמש בזה מחר בבוקר.** לא עוד שבוע. מחר. תפתח את Claude, תגיד "בוקר טוב" או "מה האימון שלי" או "תכתוב לי מייל" - ותראה איך זה מרגיש.

כי הרגע שאתה מרגיש שחסכת 20 דקות על משהו שהיה לוקח לך שעה? אתה מכור. ברמה הטובה.

## שיעור 5: "מפה לאן - המפה המלאה" (3 דקות)

### [מצלמה]

אחרון. ואולי הכי חשוב.

בנית עוזר אחד. מעולה. אבל זה רק ההתחלה.

מה שעשינו עכשיו? זה כמו שהתקנת את הטלפון הראשון שלך ופתחת אפליקציה אחת. עכשיו תדמיין מה יקרה כשתוסיף עוד.

### [מסך - מראה רשימה ויזואלית]

כל עוזר שמוסיפים? עוד 30-60 דקות חוסכים ביום. שלושה עוזרים? יש לך 2 שעות חזרה. חמישה? יש לך חצי יום.

ואתה לא עובד יותר קשה. אתה עובד חכם יותר. כי יש מישהו שעושה את הדברים שאתה לא חייב לעשות בעצמך.

## 🎬 הנחיות הקלטה

### ציוד:
- iPhone 16/17 (מצלמה)
- מחשב עם screen recording (QuickTime / OBS)
- מיקרופון (אפילו של האייפון מספיק)

### איך להקליט:
1. **חלקי מצלמה:** צלם את עצמך מדבר. רקע נקי או קו פנגאן. אור טבעי.
2. **חלקי מסך:** הקלט מסך עם QuickTime (Mac) או OBS (Windows). קרא את הטקסט בזמן שאתה עושה את הפעולות.
3. **עריכה:** חתוך את שני החלקים ביחד ב-CapCut או Remotion.

### טון:
- קז'ואל. כמו שאתה מסביר לחבר.
- אל תקרא - **תדבר**. אם צריך, תקרא פעם אחת ואז תדבר מהראש.
- אם נתקעת - תתחיל את המשפט מחדש. עורכים אחר כך.
- תחייך. תהיה אנרגטי. אתה בקו פנגאן, החיים טובים.

### Screen Recording Tips:
- הגדל את הפונט ל-150% ככה שיראו בנייד
- נקה את שולחן העבודה לפני הקלטה
- השתמש בחלון Claude נקי (בלי טאבים אחרים)
- כשמקלידים - תקליד לאט ככה שיראו מה כותבים
